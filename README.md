## What You Can Do

**Dashboard**
See your full trading performance at a glance, total PnL, today's PnL, win rate, profit factor, trade day count, current streak, and a consistency score with a live ring chart. Includes a monthly heatmap, daily bar chart, 7-day PnL row, and recent performance table, all navigable by month or week.

**Pre-Market Planning**
Build a structured plan before each session. Set market bias per asset, log economic events with impact ratings, run through mental readiness and pre-market task checklists, set a daily max loss limit, add a daily intention, and lock in the plan with a "Begin Trading Day" button. Past days are read-only for reference.

**Journal**
Log every trade with entry/exit/size/PnL/setup, attach screenshots (upload, drag-and-drop, or paste), rate execution and session quality, track emotional state, tag entries, and write out what went right, what went wrong, and the lesson learned. Entries are sortable and searchable, grouped by month.

**Reflective Journal**
Review your last 3 entries, log weekly patterns, fill out a structured weekly review (biggest mistake, best trade, discipline rating), and track earned milestones automatically.

**Study Library**
Build a personal trading knowledge base. Each lesson supports a full write-up, entry rules, key points, personal notes, image/chart uploads, and an embedded YouTube video. Filter by category or difficulty. Mark concepts as mastered.

**Playbook**
Document your setups with entry rules, SL/TP logic, invalidation conditions, session fit, and risk parameters. Tag setups as active, testing, or retired.

**Goals**
Set and track daily, weekly, monthly, and milestone goals with progress bars. Goals added from pre-market flow directly here. Includes a daily hit/miss calendar for the current month.

**Multi-Account Support**
Add multiple accounts (funded, evaluation, personal, demo) with starting balances and broker info. Switch accounts from the top bar — or aggregate all accounts into one combined view.

## Settings

| Setting | What It Does |
|---|---|
| **Name / Alias** | Your display name and trading handle |
| **Timezone** | Sets all date calculations — defaults to PST |
| **Accounts** | Add, remove, and switch between trading accounts |
| **Export / Import** | Back up and restore all data as a JSON file |
| **Clear All Data** | Wipes everything (auto-downloads a backup first) |

---

## Data & Privacy

All data is stored in your browser's `localStorage`. Nothing is sent anywhere. Exporting produces a `.json` file you can back up or move between devices.
